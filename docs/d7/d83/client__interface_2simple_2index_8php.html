Detailed Description

This file is a very simple interface that is designed to return a basic XHTML string, in response to a search. In order to use this, you need to call: <ROOT server="" base="" uri>="">/client_interface/simple/ with the same parameters that you would send to an advanced search. The results will be returned as XHTML data.

This file can be called from other servers.
