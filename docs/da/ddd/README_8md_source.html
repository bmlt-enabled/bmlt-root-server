5 DESCRIPTION
6 -----------
7 
8 The Basic Meeting List Toolbox (BMLT, hereafter) is a very powerful client/server system
9 that has been written for a very specific purpose, for a very specific clientele.
10 
11 It is designed to track and locate Narcotics Anonymous meetings, which are regularly-
12 scheduled weekly, recurring events.
13 
14 The intended clientele is Narcotics Anonymous Service bodies. They implement a BMLT
15 server, and provide the server to other NA Service bodies.
16 
17 You can find out way too much about the BMLT on [the BMLT Documentation Site](https://bmlt.app).
18 
19 [You can follow us on Twitter for release announcements](http://twitter.com/BMLT_NA).
20 
21 This project is the "root" server for the BMLT. It is the "server" part of the BMLT
22 "client/server" architecture.
23 
24 The project is actually several years old. It is currently hosted on [GitHub](https://github.com).

40 REQUIREMENTS
41 ------------
42 
43 The entire system is written in PHP, JavaScript, XHTML and CSS. Most of the code is PHP.
44 
45 PHP 5.6 or above is required to establish a root server, and 5.0 or above for a satellite.
46 
47 For more information about server requirements, see [the page on Server Requirements](https://bmlt.app/specific-topics/server-requirements/)
48 
49 INSTALLATION
50 ------------
51 
52 You set up a root server as the central database and administration area for a BMLT
53 implementation, but most people access it through what we call "satellite servers." These
54 satellites comprise the "client" part of the BMLT "client/server" architecture.
55 
56 For instructions on installing the root server, see [the page on installing a new Root Server](https://bmlt.app/installing-a-new-root-server/)
